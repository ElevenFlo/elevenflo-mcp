# Request Access

ElevenFlo MCP is available as a controlled beta.

Use it to connect Claude, Codex, Cursor, VS Code, ChatGPT, or another MCP-compatible client to ElevenFlo's bankruptcy and restructuring intelligence.

## Best fit

ElevenFlo MCP is built for:

- restructuring lawyers
- distressed investors and analysts
- investment bankers
- consultants and advisors
- journalists and market researchers

## What You Can Do

- Find the right Chapter 11 case.
- Search dockets and court filings.
- Retrieve exact filing language.
- Analyze motions, orders, plans, and disclosure statements.
- Build case briefs and internal research notes from court-grounded source material.
- Monitor case milestones, hearings, filings, and distress signals.

## Access Requirements

Beta access requires:

- an ElevenFlo account
- MCP access enabled for your organization
- a supported MCP client
- browser access for ElevenFlo sign-in and consent

## Endpoint

Use the production endpoint:

```text
https://elevenflo.com/mcp
```

## Request

Send your ElevenFlo contact:

- organization name
- account email
- intended MCP client
- expected use case
- whether access should be for one user or an organization

Before granting access, ElevenFlo may confirm permitted users, tool access, usage limits, and support contact details.
