# Setup

Start with the [Quickstart](quickstart.md), then use the client-specific setup
guide for your MCP client:

- [Claude Code](client-claude-code.md)
- [Claude Desktop](client-claude-desktop.md)
- [Codex CLI](client-codex-cli.md)
- [Cursor](client-cursor.md)
- [VS Code](client-vs-code.md)
- [ChatGPT](client-chatgpt.md)
- [Other clients](client-other.md)

Endpoint:

```text
https://elevenflo.com/mcp
```

Transport: remote MCP over HTTP / Streamable HTTP.

Use OAuth sign-in only. Do not manually paste a bearer token, API token, or
custom `Authorization` header for ElevenFlo MCP.
